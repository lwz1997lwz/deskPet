# 桌面宠物程序设计规格

## 概述

使用用户提供的猫咪照片素材，开发一个跨平台（macOS/Windows）桌面宠物程序。猫咪会在桌面上自由活动，提供陪伴感和实用功能。

## 技术栈

- **框架：** Electron（第一版）
- **渲染：** 单窗口 + Canvas 动画
- **跨平台：** 支持 macOS 和 Windows

## 核心功能

### 1. 猫咪行为系统

猫咪具有以下状态：

| 状态 | 描述 | 触发条件 |
|------|------|----------|
| IDLE（待机） | 坐着或站着，偶尔舔毛、打哈欠 | 默认状态 |
| WALK（走动） | 在活动区域内随机漫步 | 随机计时器（3-8秒） |
| SLEEP（睡觉） | 找位置趴下打盹 | 随机计时器 |
| INTERACT（点击互动） | 被点击时抬头看、叫一声 | 鼠标点击 |
| DRAG（拖拽） | 跟随鼠标移动 | 鼠标按下拖拽 |
| CHASE（追逐鼠标） | 追逐静止的鼠标 | 鼠标静止 3 秒 |

状态转换规则：
- IDLE → WALK/SLEEP（随机触发）
- IDLE → INTERACT（点击触发）
- IDLE → CHASE（鼠标静止触发）
- 任意状态 → DRAG（鼠标按下触发）
- DRAG → IDLE（鼠标释放）
- SLEEP → IDLE（点击吵醒）

### 2. 实用功能

#### 待办事项提醒

- **添加方式：** 右键猫咪 → 菜单 → "添加待办"
- **提醒方式：** 猫咪走到屏幕中央，头顶冒出气泡显示待办内容
- **查看方式：** 右键猫咪 → "待办列表"，显示浮层列表
- **交互：** 可勾选完成/删除

#### 系统资源监控

- **监控指标：** CPU 使用率、内存使用率
- **更新频率：** 每 2 秒检测一次
- **视觉表现：**
  - CPU 正常（<60%）：猫咪正常活动
  - CPU 较高（60-80%）：猫咪开始喘气
  - CPU 过高（>80%）：猫咪热得吐舌头，头顶出现 🔥 图标
  - 内存较高（>70%）：猫咪头顶冒汗，出现 💧 图标

### 3. 猫咪外观自定义

#### 素材规范

- **格式：** PNG（支持透明背景）或 WebP
- **尺寸：** 建议 128x128 或 256x256 像素，最大 512x512
- **背景：** 必须透明

#### 素材目录结构

```
assets/
└── cats/
    └── my-cat/           # 猫咪名称
        ├── idle/          # 待机动画帧
        ├── walk/          # 走路动画帧
        ├── sleep/         # 睡觉动画帧
        ├── interact/      # 互动动画帧
        └── drag/          # 拖拽动画帧
```

#### 自定义流程

1. 右键猫咪 → "设置" → "更换猫咪"
2. 点击 "导入新猫咪" 按钮
3. 选择包含素材的文件夹
4. 预览各状态动画效果
5. 确认导入

### 4. 猫咪活动范围

- **默认：** 屏幕底部区域
- **自定义：** 用户可在设置中拖拽选择猫咪的活动区域
- **边界处理：** 碰到边界自动转向

### 5. 外观定制

- **大小调整：** 滑块控制（50%-200%）
- **透明度调整：** 滑块控制（20%-100%）

## 界面设计

### 设置界面

- **入口：** 右键猫咪 → "设置" 或 系统托盘 → "设置"
- **形式：** 独立弹出窗口，非模态
- **分类：**
  - 外观设置（大小、透明度、猫咪选择）
  - 行为设置（活动范围、走动速度、追逐鼠标开关）
  - 功能设置（系统监控开关、阈值调整）
  - 通用设置（开机自启、置顶显示）

### 系统托盘菜单

- 📋 添加待办...
- ⚙️ 设置
- 🔄 重启猫咪
- ❌ 退出

### 右键菜单

- 👋 摸摸头
- 📋 添加待办...
- 📝 待办列表
- 🔄 换只猫咪
- ⚙️ 设置
- 😴 让我静静（隐藏）

## 视觉风格

- **图标风格：** Emoji 风格（第一版）
- **气泡样式：** 毛玻璃效果
- **菜单样式：** 深色毛玻璃风格
- **整体色调：** 温馨可爱

## 架构设计

```
│  └─────────────┘  └─────────────┘  └─────────────┘        │
└─────────────────────────────────────────────────────────────┘
                           │
                    IPC 通信 (ipcMain)
                           │
┌─────────────────────────────────────────────────────────────┐
│                    渲染进程                                  │
│  ┌─────────────────────────────────────────────────────┐   │
│  │              Canvas 画布（透明窗口）                   │   │
│  │  ┌─────────┐  ┌─────────┐  ┌─────────┐             │   │
│  │  │ 猫咪精灵 │  │ 特效层  │  │ UI 浮层 │             │   │

│  │ 动画引擎    │  │ 状态管理    │  │ 事件系统    │        │
│  └─────────────┘  └─────────────┘  └─────────────┘        │
└─────────────────────────────────────────────────────────────┘
```

### 核心模块

#### 渲染进程模块

1. **Canvas 画布**
   - 透明无边框窗口
   - 多层渲染：背景层 → 猫咪层 → 特效层 → UI层
   - requestAnimationFrame 驱动

2. **猫咪精灵系统**
   - 状态机管理（走动/睡觉/待机/互动）
   - 帧动画播放器
   - 位置/速度/方向控制

3. **交互系统**
   - 鼠标点击/拖拽检测
   - 碰撞检测（窗口边界）
   - 鼠标追逐逻辑

#### 主进程模块

1. **窗口管理**
   - 透明无边框窗口创建
   - 窗口层级控制（置顶）
   - 鼠标穿透控制

2. **系统集成**
   - 系统托盘菜单
   - 开机自启设置
   - 全局快捷键

3. **数据持久化**
   - 用户配置存储
   - 待办事项存储
   - 猫咪状态保存
